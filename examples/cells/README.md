# Example HoneyHex cells

These directories illustrate **conventions** for different agent workflows. They are **not** included in the PyPI package (`pip install honeyhex`); clone this repository to browse them.

| Directory | Focus |
|-----------|--------|
| [game-dev](game-dev/) | Milestones, builds, design notes |
| [research](research/) | Papers, hypotheses, experiment log |
| [coding-agent](coding-agent/) | Tasks, refactors, review notes |

Common pattern:

1. Treat the sample folder as a cell root (or copy `.honeyhex` ideas into your project).
2. Run `hex cell init --guided` for a quick first commit, or follow [Getting started](../../documentation/getting-started.md).
